# Snake Game (Java Swing)

A classic Snake game built using Java and Swing, featuring real-time gameplay, keyboard controls, collision detection, and score tracking. The project demonstrates core concepts of object-oriented programming, event-driven programming, and GUI development in Java.

## Features

- Real-time game loop using Java Swing Timer
- Keyboard-controlled movement (Arrow Keys)
- Collision detection (walls and self-collision)
- Dynamic score tracking
- Randomized food generation
- Game over state handling

## Controls
- Arrow Keys → Move Snake (Up, Down, Left, Right)

## Tech Stack
- Java
- Java Swing (GUI)
- AWT (Graphics & Event Handling)

## Project Purpose
Built to strengthen understanding of object-oriented programming, GUI development, and game logic implementation in Java.

## Note
This is a standalone desktop application and does not require external libraries. It is intended as a learning project to practice core Java concepts and game development logic.
